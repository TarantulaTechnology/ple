This text covers all the usual programming language concepts, including binding, scope, parameterization, modularity, concurrency, and meta­programming, not with a dry summary of each concept, but rather within the context of whole-language overviews with plenty of runnable example scripts. We aim to help developers learn a bit of theory, and introduce students to a number of modern languages. Check out the source code for the examples given in the book.

Additional Exercises

The text features 300 exercies, most of which are “further explorations” in which you find or install a language interpreter, investigate aspects of the language not covered in the chapter proper, muse about language designers’ intents, and try your hand at both large and small programming exercises. If you would like to contribute additional exercises, submit a pull request for this file.
