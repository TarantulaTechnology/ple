This text covers all the usual programming language concepts, including binding, scope, typing, parameterization, modularity, concurrency, and meta­programming, not via a dry summary of each concept, but rather within the context of whole-language overviews with plenty of runnable example scripts. We aim to help developers learn a bit of theory, and introduce students to a number of modern languages. All of the source code for the book’s examples, as well as number of bonus scripts, can be found in this GitHub repository.

Good to Know

This book is not for beginning programmers. We’re assuming you know at least two languages pretty well, and hopefully a couple more. You should know the basic concepts surrounding variables, expressions, operators, functions, and basic data structures, and have experience writing nontrivial applications—this book is a tour, not a tutorial. We cover many interesting (and often powerful) features, and purposely do so with code that is often very dense and sometimes cryptic, even when presenting unfamiliar paradigms for the first time.

Most of the code examples are written as their own unit tests, which should help you learn the concepts as you read the code itself. We favor this style over comments, which, you know, can become stale and misleading. Uncle Bob tells us why comments are not “pure good.”

Additional Exercises

The text features 300 exercises, most of which are “further explorations” in which you find or install a language interpreter, investigate aspects of the language not covered in the chapter proper, muse about language designers’ intents, and try your hand at both large and small programming tasks. If you would like to contribute additional exercises, submit a pull request for this file.
