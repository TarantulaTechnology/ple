# Clojure Examples

To get Clojure:

```
brew install leiningen
```

To run the tests:

Create a script called `clj` to invoke Clojure directly and
put it in your path. Mine looks lke:
`java -cp ~/lib/clojure-1.8.0.jar clojure.main $*`. Then


```
./test.sh
```
